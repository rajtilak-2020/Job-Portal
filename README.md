# Job Listing Portal

A dynamic web application connecting job seekers with employers efficiently.

## Description

The Job Listing Portal provides a clean and intuitive interface for job searching and recruitment. Job seekers can search openings, create profiles, and apply directly. Employers can post detailed listings and manage applications. Features include advanced search filters, real-time notifications, and secure authentication.

## Features

### User Authentication
- User registration and login
- Secure password storage and authentication

### Job Listings
- Create, edit, and delete job postings
- Include title, description, qualifications, responsibilities, location, and salary range

### Job Search
- Simple search functionality with keyword support
- Filters for job type, location, and more

### Profile Management
- **Job Seekers**: Personal information, resume upload, contact details
- **Employers**: Company information and contact details

### Job Application
- Direct application through the portal
- Application tracking and candidate management

### Dashboard
- **Job Seekers**: Track applied jobs and update profiles
- **Employers**: Manage listings and view applications

## Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd Job-Portal

# Install dependencies
npm install

# Start the application
npm start
```

## Usage

1. Register as a job seeker or employer
2. Complete your profile
3. Job seekers: Search and apply for jobs
4. Employers: Post jobs and manage applications
